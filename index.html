Function Cheatsheet

Regular Functions

Arrow Functions

Regular function with no return keyword


      function f(){
        //returns undefined
      }

Arrow function with no return keyword.


      const f = () =>{
        //returns undefined
      }

Regular function with return keyword.


      function f(){
        return true;
      }

Arrow function with return keyword inside curly braces.


      const f = () => {
        return true;
      }

Arrow function with return value but no curly braces.


      const f = () => true;

Regular function with single parameter and return value.


      function f(name){
        return name.toUpperCase();
      }

Arrow function with single parameter and return value.


      const f = name => {
        return name.toUpperCase();
      }

Arrow function with single parameter and return value but no curly braces. 


      const f = name => name.toUpperCase();

Regular function with multiple parameters and default values, returning true. 


      function f(isAlive=true, department=42){
        return true;
      }

Arrow function with multiple parameters and default values, returning true. 


      const f = (isAlive=true, department=42) => true;

Regular IIFE that accepts parameters.


      (function f(name, message){
        console.log(`${name} says ${message}`);
      })('Sheldon', 'Bazinga');

Arrow IIFE that accepts parameters.


      ((name, message)=>{
        console.log(`${name} says ${message}`);
      })('Sheldon', 'Bazinga');

Regular function returning an object.


      function f(){
        return {
          id: 123,
          name: 'Steve'
        }
      }

Returning an object on the next line requires parentheses.


        function f(){
          return (
            {
              id: 123,
              name: 'Steve'
            }
          )
        }

Arrow function returning an object.


        const f = () =>{
          return {
            id: 123,
            name: 'Steve'
          }
        }

Arrow function returning an object in single line requires parentheses. 


        const f = () =>( { id: 123, name: 'Steve' } );

Regular function destructuring an object as a parameter.


      function f({id, name}){
        //id and name are now local variables inside this function
      }

      let myObj = {
        id: 123,
        name: 'Steve'
      }
      f(myObj);

Arrow function destructuring an object as a parameter.


      const f = ({id, name}) =>{
        //id and name are now local variables inside this function
      }

      let myObj = {
        id: 123,
        name: 'Steve'
      }
      f(myObj);
